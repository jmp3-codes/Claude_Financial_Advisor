SETUP INSTRUCTIONS FOR EXCEL FILE CONVERSION
==============================================

Due to project permission constraints, binary files need special handling.

SOLUTION:
---------
1. Download the Python script from this location (or copy-paste the code)
2. Run it from your terminal:

   python3 ~/Documents/Coding/create_xlsx_files.py

OR

1. Open Excel manually
2. File > New > Create blank workbooks
3. Save them as:
   - /Users/jackpage/Documents/Coding/Claude/Financial Advisor/master_files/merchant_dictionary.xlsx
   - /Users/jackpage/Documents/Coding/Claude/Financial Advisor/master_files/master_transactions_2025.xlsx

Then set up the sheets as follows:

MERCHANT DICTIONARY Columns:
- Raw Statement Text
- Business Name
- Category
- Subcategory
- Notes
- Last Seen
- Frequency

MASTER TRANSACTIONS Sheets:
Sheet 1: "All Transactions"
  Columns: Transaction ID, Date, Account Source, Raw Description, Business Name, Category, Subcategory, Amount, Transaction Type, Statement Period, Calendar Month, Notes, Verified

Sheet 2: "Monthly Summary"
  Columns: Month, Total Income, Total Expenses, Net Cash Flow, Housing, Transportation, Food & Dining, Healthcare, Entertainment, Shopping, Personal Care, Financial, Insurance, Subscriptions, Travel, Education, Gifts & Donations, Pets, Other

Sheet 3: "Category Analysis"
  Columns: Category, Jan, Feb, Mar, Apr, May, Jun, Jul, Aug, Sep, Oct, Nov, Dec, YTD Total, Monthly Avg

Sheet 4: "Annual Dashboard"
  Content: 2025 Financial Summary (title in cell A1)
